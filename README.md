Скрипт для автоустановки сервера XRAY + REALITY.
Автоустановится сам сервре, бот в ТГ для панели управления с необходимыми фичами.
Также выдается первая ссылка на профиль, далее профили создаются в боте.

Установка на чистом серваке: apt update -y && apt install git -y && git clone https://github.com/M4ryJ4ne/xray-panel.git && cd xray-panel && ./install_xray.sh
Под конец установщик запросит такие данные:
Имя первого профиля.
Токен бота ТГ.
Пароль для этого бота (на первое подключение).
Пароль для кнопки "Перезапуск сервера" (чтобы случайно не ребутнуть кнопкой).
Дата оплаты сервера (для уведомления в бота).
